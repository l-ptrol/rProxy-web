# rProxy v6.0 (Python Core Edition)

Мощная модульная система управления SSH-туннелями и VPS публикациями, полностью переписанная на Python.

## Новое в v6.0
- **Modular Core**: Логика отделена от интерфейсов. Всё управление (CLI и Web) работает на едином Python-ядре.
- **Python CLI**: Команда `rproxy` теперь — это мощное Python-приложение, заменяющее оригинальный bash-скрипт.
- **Premium Web**: Тот же "стеклянный" дизайн (Glassmorphism), но теперь с мгновенной скоростью работы через прямое обращение к ядру.
- **Стабильность**: Отказ от хрупких `sed/grep` в пользу надежной работы с объектами данных в Python.

## Установка
```bash
sh <(curl -sL https://raw.githubusercontent.com/l-ptrol/rProxy-web/master/install.sh)
```

## Использование
### В терминале:
Просто введите:
```bash
rproxy
```
(скрипт установки автоматически создаст симлинк).

### В браузере:
1. Запустите веб-службу: `/opt/etc/init.d/S99rproxy-web start`
2. Откройте: `http://адрес-роутера:3000`

## Архитектура
- `core/`: Ядро системы (логика).
- `rproxy.py`: Консольный интерфейс.
- `main.py`: Веб-интерфейс.
- `templates/`: Файлы дизайна.
