# rProxy Web Dashboard (Python Reborn)

Легковесный и современный веб-интерфейс для управления SSH-туннелями и обратным прокси через скрипт `rProxy`. Специально оптимизирован для роутеров Keenetic (Entware).

## Особености (v2.0.1 Python)
- **Ultra-Light**: Минимум зависимостей, отказ от тяжелого Node.js.
- **FastAPI + HTMX**: Мгновенный отклик интерфейса без перезагрузки страниц.
- **Neon Glass Design**: Премиальный дизайн с эффектом матового стекла и неоновой индикацией.
- **Мобильная совместимость**: Идеальное отображение на смартфонах.
- **Прямое управление**: Запуск, остановка и удаление сервисов в один клик.

## Установка (для Keenetic / Entware)

Выполните команду в терминале роутера:

```sh
curl -sL https://raw.githubusercontent.com/l-ptrol/rProxy-web/master/install.sh | sh
```

### После установки:
1. Запустите службу: `/opt/etc/init.d/S99rproxy-web start`
2. Откройте в бразуере: `http://IP_ВАШЕГО_РОУТЕРА:3000`

## Требования
- Установленный [Entware](https://help.keenetic.com/hc/ru/articles/115000021505)
- Установленный основной скрипт [rProxy](https://github.com/l-ptrol/rProxy)
