# rProxy Web Premium Dashboard v5.0

Суперсовременный, легкий и быстрый веб-интерфейс для управления скриптом rProxy на роутерах Keenetic.

## Особенности
*   **Premium Glassmorphism**: Эффект матового стекла, динамические градиенты и микро-анимации на чистом CSS.
*   **Легкость**: Написано на Python + Bottle. Никакого Node.js, React или тяжелых фреймворков.
*   **Мобильность**: Полная адаптивность под вертикальные экраны смартфонов.
*   **Родная интеграция**: Работает напрямую с конфигами `rproxy` через Entware.

## Установка
Выполните команду в консоли роутера:
```bash
sh <(curl -sL https://raw.githubusercontent.com/l-ptrol/rProxy-web/master/install.sh)
```

## Запуск
```bash
/opt/etc/init.d/S99rproxy-web start
```
Интерфейс будет доступен на порту **3000**.
